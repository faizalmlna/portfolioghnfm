Stuktur data dan Array

  • admin
  • July 07, 2023

Struktur data merupakan cara-cara atau metode yang digunakan untuk menyimpan data di dalam memori komputer, Salah satu struktur data yang sering digunakan dalam pemrograman adalah Array, Array merupakan struktur data yang digunakan untuk menyimpan sekumpulan data dalam satu tempat.

example membuat array pada Javascript

< script >
  var products = ["senter", "Radio", "Antena", "Obeng"];

  document.write(<h3>Daftar Produk : </h3>
  document.write("<ol>")

  products.forEach((data ) =>{
  document.write('<li>${data}</li>');
})
  document.write("</ol>")
< /script >

Menambahkan Data ke Dalam Array

Ada dua cara yang bisa dilakukan untuk menambah data ke dalam array yaitu dengan mengisi menggunakan Indeks; dan menggunakan method push().

example menambahkan menggunakan push

< script >
  var products = ["senter", "Radio", "Antena", "Obeng"];

  products.push("Televisi");

  document.write("products");
< /script >

Menghapus Data Dari Array

Seperti menambahkan data ke array, menghapus data juga memiliki dua cara yaitu dengan delete dan menggunakan method pop(), method akan menghapus arrat yang ada dipaling belakang. arrat pada Javascript dapat dikatakan sebagai sebuah stack (tumpukan), yang mana memiliki sifat LILO (Last In Last Out). apa bila kita ingin menghapus data pada indeks tertentu, maka fungsi atau method yang digunakan adalah splice().

Fungsi ini memiliki dua parameter yang harus diberikan:

array.splice(<indeks>,<total>);

Keterangan

  • <indeks> adalah indeks dari data didalam array yang akan dihapus.
  • <total> adalah jumlah data yang akan dihapus dari indeks tersebut.

Mengubah Isi Data Pada Array

untuk mengubah isi data pada array bisa menggunakan script sebagai berikut

< script >
  var products = ["senter", "Radio", "Antena", "Obeng"];

  products[1] = "Komputer";

  document.write("products");
< /script >

Method-method Array Lain

  1. Method Filter()

    2. Method filter() berfungsi untuk menyaring data dariarray. Parameter yang harus diberikan pada method filter() sama seperti method forEach(), yaitu: sebuah fungsi callback.

    < script >
      const angka = [1, 2, 3, 4, 5, 6, 7, 8, 9];

      consr filteredArray = angka.filter((item)=> {
      return item % 2==0});

      document.write(filteredArray);
    < /script >

  2. Method Includes()

    3. Method ini berfungsi untuk mengecek apakah sebuah data ada di dalam array atau tidak. Biasanya digunakan untuk melakukan pencarian untuk memastikan data sudah ada di dalam array.

    < script >
      var tanaman = ["padi", "kacang", "Jagung", "Kedelai"];
      var adakacang = tanaman.includes("kacang")
      document.write(adakacang);
    < /script >

  3. Method Sort()

    4. Metheod sort() berfungsi untuk mengurutkan data pada array

    < script >
      var alfabet = ['a', 'b', 'e', 'd', 'c', 'f', 'g'];
      var angka = [3,2,1,4,6,8]

      document.write(alfabet.sort());

      document.write(angka.sort());
    < /script >
